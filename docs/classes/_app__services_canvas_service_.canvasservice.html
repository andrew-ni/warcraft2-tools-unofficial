Class CanvasService

CanvasService manages drawing output of the map to the map display window. It is the View component of this MVC model.

Private drawImage

  • drawImage(layer: CanvasRenderingContext2D, image: ImageBitmap, player: number, width: number, pos: Coordinate, index: number): void
    • Defined in app/_services/canvas.service.ts:357

    Used to draw terrain, units, and assets onto the canvas.

    Parameters

    • layer: CanvasRenderingContext2D

      The canvas context to draw on.

    • image: ImageBitmap

      Image stored in assetMap. Access with assetMap.get(imgName)

    • player: number

      Player 1 - 8. Used to calculate x offset in source image.

    • width: number

      Width of a single sprite on the image.

    • pos: Coordinate

      X, Y coordinate to draw on canvas.

    • index: number

      Position in the spritesheet. Used to calculate y offset in source image. Starts at 0. void ctx.drawImage(image, sx, sy, sWidth, sHeight, dx, dy, dWidth, dHeight);

    Returns void
